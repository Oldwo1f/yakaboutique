
********************************
Description
********************************
Ce module permet d'assurer la livraison des commandes en magasin / point de ventes. 
A l'aide d'une google Maps le client pourra choisir le point de retrait de la commande le 
plus près de chez lui.


********************************
Les +
********************************
Le client aura le choix entre une livraison par un transporteur normal ou de se 
faire livrer directement en magasin. Le module se base sur un transporteur de votre
choix pour affecter la livraison en magasin. Le fait de l'affecter à un transporteur 
permet de conserver les tarifs de livraison / délai...
L'intérêt est que l'adresse de livraison du magasin sera renseigné sur la commande, 
la facture, le bon de livraison... Sur la google maps votre client pourra aussi retrouver
toutes les coordonnées magasins ainsi que les horaires d'ouverture.


********************************
Installation
********************************
Ce module est uniquement valable pour la version 1.5 et supérieure de Prestashop.
Ce module est déjà compatible avec la version 1.6.0.4 (RC) de Prestashop

Installer votre module normallement. L'administration en back Office se trouve sur 
le bouton configurer de votre module.

Aucune autre action d'installation n'est requise de votre part 


********************************
Fonctionnement
********************************
1 : Vous installez le module
2 : Vous créez un transporteur que vous pourrez par exemple nommer "Livraison en point de vente" ou "Retrait en magasin"...
3 : Dans l'administration du module vous définissez le transporteur
3 : (Optionnel) : Dans l'administration du module vous définissez si chaque magasin 
doit recevoir un email lorsqu'une commande est passée. Pour cela les emails des fiches 
magasins doivent être remplit
4 : (Optionnel) : Dans l'administration du module vous pouvez choisir d'afficher ou non
une google map des magasins.
5 : (Optionnel) : Hauteur de la Google Map
6 : (Optionnel) : Largeur de la Google Map


********************************
Fonctionnalités
********************************
- Compatible avec le paiement en 1 page (One Page checkout).
- Compatible Multilangue
- Compatible Multiboutique
- L'adresse du magasin est indiquée dans la commande, la facture, le bon de 
livraison... en tant qu'adresse de livraison.
- Possibilité d'activer une google maps pour présenter la liste des magasins
- Affichage sur la Google maps des coordonnées magasin et des horaires d'ouverture
- Possibilité de règler les dimensions de la google map
- Possibilité d'envoyer un email à chaque magasin indépendemment pour les prévenir qu'une 
commande vient d'être passée
